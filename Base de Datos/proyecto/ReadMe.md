# Desarrollo de formulario en HTML y Base de datos

En esta carpeta se encuentra el desarrollo de un formulario en html y php. El formulario se encarga de conectarse a la base de datos y poder almacenar archivos .txt que corresponden a electroencefalogramas. Para poder utilizar la base de datos y el formulario, es necesario contar con las herramientas descritas [aquí](https://github.com/larivera-UVG/Datos-Epilepsia/blob/master/Base%20de%20Datos/ReadMe.md).
